# Stock Profit using Binary Search

A simple program to calculate maximum profit from stock prices using the **binary search** approach.

This project takes an array of stock prices (for example dailty prices) and finds the **best buy-sell profit** using a binary search based technique.

---

## 🚀 Features

- Computes maximum possible profit from stock prices
- Uses efficient searching (binary search) logic to reduce time complexity
- Easy to understand and modify

---

## 📝 Description

This project demonstrates how **binary search** can be applied to the problem of finding maximum profit in a stock price list. Rather than checking all possibilities, the algorithm efficiently narrows down the search to find the optimal buy-sell pair.
